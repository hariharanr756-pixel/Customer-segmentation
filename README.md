# Customer Segmentation Project

## 📌 Objective

Segment customers based on annual income and spending score using K-Means clustering to identify different customer groups.

---

## 🛠 Tools & Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📊 Project Steps

1. Import required libraries
2. Load dataset (Mall_Customers.csv)
3. Data exploration and cleaning
4. Apply Elbow Method to find optimal clusters
5. Perform K-Means clustering
6. Visualize customer segments

---

## 📈 Result

Customers are grouped into distinct clusters based on their income and spending behavior.
This helps businesses target customers effectively and improve marketing strategies.

---

## 📌 Conclusion

K-Means clustering successfully identified different customer segments.
Businesses can use this segmentation to make data-driven decisions and increase profitability.
