# Customer Segmentation Project

## Objective
Segment customers based on income and spending behavior using K-Means clustering.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps
1. Data Loading
2. Data Analysis
3. Elbow Method
4. K-Means Clustering
5. Visualization

## Result
Customers were grouped into clusters based on income and spending score.
